### Evaluating AGENTS.md Are Repository-Level Context Files Helpful for Coding Agents?

https://arxiv.org/html/2602.11988v1#S6
挺尴尬的一篇研究，让 Agent 自己读库总结的详尽 AGENTS.md 原来只是人类的一厢情愿
其实真实效果并不好，几乎无提升的同时，token 消耗还提升了不少。难怪之前看到说 Coding Agent 其实有时候会可以忽略 AGENTS.ms ，效果会有点滥用
建议是 AGENTS.md 只写核心原则，可以仓库无关
